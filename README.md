# 원자 모형 퀴즈

스마트폰으로 참여하는 공개수업용 원자 모형 퀴즈 웹앱입니다.

## 기능

- 1~20번 원소 중 규소를 제외하고 3문항 랜덤 출제
- 사진 또는 터치 캔버스 중 하나로 결과물 제출
- 같은 조 결과와 전체 조 결과 확인 및 동료 피드백
- 학생 본인 답안 조회와 관리자 승인 1회 수정
- 관리자 제출자 팝업, 공개 여부, 다운로드, 전체 삭제
- 관리자 설정으로 정답 확인과 원소 검색 활성화/비활성화

## Firebase 설정

1. Firebase Console의 Authentication에서 `익명`과 `이메일/비밀번호` 로그인을 활성화합니다.
2. 관리자 Gmail 계정을 만들고 UID가 `E2aywHHeSvbpcVW81AxcYIrubvE2`인지 확인합니다.
3. 승인된 도메인에 `ppaperly.github.io`를 추가합니다.
4. Firestore Rules 화면에 저장소의 `firestore.rules`를 게시합니다.

관리자 화면에는 Gmail 아이디 부분과 비밀번호만 입력합니다. `@gmail.com`은 자동으로 붙으며 비밀번호는 코드나 Firestore에 저장하지 않습니다.

## 수업 운영

학생은 사진 또는 캔버스를 선택해 제출합니다. 퀴즈 진행 중 뒤로가기를 누르고 취소를 확정하면 미완료 참여 기록과 이미 제출한 결과물이 삭제됩니다.

관리자는 참여 기록을 선택해 답안을 한 문제씩 보고, 수정 권한 부여, 다른 학생 조회 가능 여부, 그림 다운로드와 전체 삭제를 관리합니다.
