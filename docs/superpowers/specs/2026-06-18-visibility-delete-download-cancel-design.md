## 관리자 전체 삭제

- 관리자 명단에 참여 기록 삭제 버튼을 제공한다.
- 확인창에 학생 정보와 세션·제출물·피드백 전체가 삭제됨을 표시한다.
- 확인 후 해당 세션, 연결된 제출물 3개, 각 제출물의 피드백을 Firestore 배치로 삭제한다.
- 완료된 기록과 미완료 기록 모두 관리자만 삭제할 수 있다.
- 일부만 삭제된 상태가 남지 않도록 한 배치에서 처리한다.

## 제출 그림 다운로드

- 관리자 답안 팝업에서 현재 문제의 사진 또는 캔버스 JPEG를 개별 다운로드한다.
- 파일명은 `학년숫자반숫자번호이름(문제번호).jpg`이다.
- 예: 2학년 1반 10번 홍길동의 1번 문제는 `2110홍길동(1).jpg`이다.
- 파일명에 사용할 수 없는 문자는 제거한다.

## 퀴즈 진행 취소

- 학생이 3문제 완료 전 브라우저 뒤로가기를 누르면 `정말 취소하시겠습니까?` 확인창을 표시한다.
- 계속하기를 선택하면 현재 입력, 사진, 캔버스 상태를 유지한다.
- 취소 확정 시 학생 본인의 미완료 세션, 이미 제출된 답안·그림, 연결 피드백을 모두 삭제한다.
- 삭제 성공 후 학생 정보 입력 화면으로 이동한다.
- 학생은 완료된 참여 기록이나 다른 학생 기록을 삭제할 수 없다.
