# 관리자 제출자 카드 명단 설계

## 목표

관리자 제출자 명단을 학년·반 복수 필터와 세 가지 배치 방식으로 조회하고, 스마트폰 2열·PC 4열의 정사각형 카드로 표시한다.

## 필터

- 기본 상태는 전체 학년·전체 반 조회이다.
- 데이터에 존재하는 학년과 반을 체크박스로 자동 생성한다.
- 학년과 반은 각각 복수 선택할 수 있다.
- 아무 체크박스도 선택하지 않으면 해당 항목은 전체로 처리한다.
- `전체 조회` 버튼은 학년·반 선택을 모두 해제한다.
- 기존 학생 검색 입력은 함께 유지한다.

## 배치

- 기본은 `전체 배치`이다.
- 전체 배치는 학년·반별 독립 구역을 만들고, 구역 안에서 최근 참여순으로 표시한다.
- 조별 배치는 학년·반 안에서 조별 독립 구역을 만든다. 한 조의 카드 그리드가 끝난 뒤 다음 조가 새 줄에서 시작한다.
- 번호순 배치는 학년·반별 독립 구역 안에서 조와 관계없이 학생 번호 오름차순으로 표시한다.
- 동일 학생의 반복 참여는 별도 카드로 유지한다.

## 카드

- 스마트폰은 한 줄 최대 2개, PC는 한 줄 최대 4개이다.
- 카드 비율은 정사각형이며 학년, 반, 조, 번호, 이름, 제출 수와 조회 상태를 표시한다.
- 카드 본문을 누르면 기존 답안 팝업을 연다.
- 오른쪽 위 `⋯` 버튼을 누르면 `조회 여부 변경`과 `전체 삭제` 메뉴를 표시한다.
- 메뉴 바깥을 누르거나 다른 메뉴를 열면 기존 메뉴를 닫는다.
- 전체 삭제는 기존 확인창과 전체 데이터 삭제 동작을 그대로 사용한다.

## 테스트

- 체크박스 미선택 시 전체 조회
- 학년·반 복수 선택의 교차 필터
- 전체 배치의 학년·반 구역과 최근 참여순
- 조별 배치의 조 구역 분리
- 번호순 배치의 숫자 오름차순
- 카드·더보기 메뉴·배치 버튼 UI 계약
- 스마트폰 2열과 PC 4열 CSS 계약
